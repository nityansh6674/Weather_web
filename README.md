# 🌦️ Weather Forecast App

A responsive Weather Forecast application built with **React.js**, **Vite**, and the **OpenWeather API**. Users can search for any city to view real-time weather conditions along with forecast information in a clean and intuitive interface.

## ✨ Features

- 🌍 Search weather by city name
- 🌡️ Current temperature
- ☁️ Weather condition with icons
- 💧 Humidity information
- 💨 Wind speed
- 📅 Multi-day weather forecast
- 📱 Fully responsive design
- ⚡ Fast performance with Vite

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/nityansh6674/weather-app.git
```

Navigate to the project

```bash
cd weather-app
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
